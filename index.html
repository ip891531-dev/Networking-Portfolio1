Networking Fundamentals

Building a Strong Foundation in Computer Networks

What is a Computer Network?

A computer network is a collection of computers, servers, and other devices connected together to share resources and exchange data.

Types of Networks

  • LAN (Local Area Network): Covers a small area such as a home, classroom, or office building.
  • WAN (Wide Area Network): Covers large geographic areas — the Internet is the largest WAN.
  • WLAN (Wireless Local Area Network): A wireless version of a LAN using Wi-Fi.

Key Devices

  • Client: End-user device (laptop, phone) that requests information.
  • Server: Provides resources and services to clients.
  • Router: Connects multiple networks and routes data efficiently.
  • Switch: Connects devices within the same network.
  • Access Point: Allows wireless devices to join the network.

Important Vocabulary

  • Protocol: Rules that define how devices communicate.
  • Bandwidth: Maximum data transfer rate of a connection.
  • Topology: The physical or logical layout of the network.

IP Addressing & MAC Addresses

How Devices Are Identified on a Network

IP Addresses

An IP address is a unique identifier assigned to every device on a network, similar to a mailing address.

  • IPv4: 32-bit address (e.g., 192.168.1.1) — most widely used.
  • IPv6: 128-bit address (e.g., 2001:db8::1) — solves address exhaustion.

MAC Addresses

A MAC address is a permanent physical address burned into the network card. It is 48 bits long and shown in hexadecimal. The first 3 octets (OUI) identify the manufacturer.

Examples

Private IP: 192.168.1.50 (used inside local networks)
Public IP: Visible to the internet, assigned by ISP.
